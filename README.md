## What This Repo Does

This repository follows the survey framework developed for LLM agent memory safety:

- `memory ecology`: memory forms determine where risks reside
- `lifecycle governance`: memory operations determine how risks are created, amplified, and translated into behavior
- `future shift`: memory is evolving from storage infrastructure toward behavioral infrastructure
